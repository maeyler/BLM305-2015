### Most Important Java Frameworks
* Collections (List, Set, Map) -- java.util
* Threads & Time measurement -- java.lang
* I/O (Streams and Readers) -- java.io
* GUI (JComponent) -- javax.swing
* Image processing -- java.awt.image
